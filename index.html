About this site

Made with plain HTML/CSS/JS — perfect for GitHub Pages.

  • Sticky header + responsive nav
  • Two mini-games (easy to expand)
  • Leaderboard + stats stored in localStorage
  • No libraries, no build steps

Want to add more games? Copy the pattern: build a card, write logic in script.js, and award points via awardPoints().
